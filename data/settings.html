PILIH MOTOR

// preset library

Pilih motor untuk auto-config trigger, advance map, rev limit, dwell, dll. Setting custom yang sudah ada akan ditimpa.

Band antara utama dan batas maks dikelola oleh mode cut yang dipilih di bawah. Di atas batas maks selalu hard cut (paksa disarm 3 cycle) untuk safety.

[ pembatas rev · mode cut ]

—

Soft Retard: kurangi advance° di soft band → power drop halus, mesin tetap fire.
Hard Cut: paksa disarm di soft band → mesin mati total di limit.
Pattern Cut: fire X siklus lalu skip Y → suara "memantul" khas drag race.
Progressive: probabilitas skip naik linear dari 0% (di utama) → 95% (di batas maks).

[ backfire · burble · anti-lag ]

—

⚠ Backfire memperpendek umur exhaust valve & header (temperatur naik). OFF by default. Pakai dengan pertimbangan — efek suara/api, bukan power gain.

[ launch control · 2-step ]

Input GPIO13 (pull-up). Sambungkan ke saklar di tuas kopling / launch switch. Saat input LOW (kopling ditarik), pembatas rev override ke launch_rpm dengan HARD CUT.

[ quickshifter · clutchless up-shift ]

Input GPIO14 (pull-up + interrupt FALLING). Sensor strain/microswitch di tuas perseneling. Trigger → cut ignition selama cut_ms → torsi drop → gigi pindah mulus.

[ shift light · gpio27 ]

LED di GPIO27 mati di bawah warn, flash 5 Hz antara warn & shift, solid ON di atas shift. Sambungkan LED 3 mm + resistor 220 Ω dari GPIO27 ke GND.

[ kurva dwell · rpm-kompensasi ]

Bila aktif, dwell µs dihitung dari curve di bawah berdasarkan RPM saat ini. Bila nonaktif, dwell fix dari tuning manual di card sebelah dipakai.

[ tuning output pengapian ]

Dwell = berapa lama GPIO25 HIGH per tembakan (= waktu pengisian kapasitor di HV stage). Offset menambah ke setiap lookup advance° — pakai untuk kompensasi propagation delay melalui optocoupler + driver SCR (+ve = spark lebih cepat = kompensasi delay positif).

[ wiring hv stage ] 
ESP32 GPIO25 ──[ 220 Ω ]──┬── opto LED anode ── │ ── opto collector ── SCR gate
                          │                     │              │
                         GND                  opto LED      SCR cathode
                                                emitter        │
                                                              GND (HV)

  HV stage: 12V → step-up → cap 300V → SCR discharge → koil primer

  Spec GPIO25: logic 3.3 V, ~20 mA maks. Pakai resistor 220 Ω untuk
  membatasi arus LED optocoupler.

  GND ESP32 harus terhubung ke GND signal HV stage (bukan GND power koil)
  via kabel tipis tunggal untuk menghindari ground loop.

  TEST LOAD sebelum konek HV beneran: 10 kΩ + LED dari GPIO25 ke GND.
  LED harus kedip terlihat setiap Bench test fire.
[ alvp · proteksi voltase rendah ]
VBAT — V NORMAL

VBAT diukur via ADC GPIO32 + voltage divider 1:4 (mis. 30 kΩ + 10 kΩ). Di bawah derate threshold (default 10.5V), main rev limit auto-clamp ke derate_rpm (rider terasa power drop). Di bawah disarm threshold (default 9.0V), spark auto-disarm — koil HV butuh voltase cukup. Hysteresis 2 detik anti flicker saat cranking dip.
