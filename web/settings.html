Pilih motor untuk auto-config trigger, advance map, rev limit, dwell, dll. Setting custom yang sudah ada akan ditimpa.

Band antara utama dan batas maks dikelola oleh mode cut yang dipilih di bawah. Di atas batas maks selalu hard cut 100% (paksa disarm 3 cycle) untuk safety overrev — burn-through tidak berlaku di sini.

[ pembatas rev · mode cut ]

Cara firmware membatasi RPM saat melebihi limit. Soft halus (retard timing), Hard mati total, Pattern skip pola tetap, Progressive skip random. Pilih sesuai feel yang diinginkan saat menyentuh limiter.

—

Soft Retard: kurangi advance° di soft band → power drop halus, mesin tetap fire.
Hard Cut: paksa disarm di soft band → mesin mati total di limit.
Pattern Cut: fire X siklus lalu skip Y → suara "memantul" khas drag race.
Progressive: probabilitas skip naik linear dari 0% (di utama) → 95% (di batas maks).

[ backfire · burble · anti-lag ]

Bikin suara pop/dor di knalpot saat decel/idle dengan retard timing ekstrem. Efek audio + flame di header. Memperpendek umur exhaust valve — pakai bijak.

—

⚠ Backfire memperpendek umur exhaust valve & header (temperatur naik). OFF by default. Pakai dengan pertimbangan — efek suara/api, bukan power gain.

[ idle rumble · lopey cam / drag-burble ]

Bikin suara "lopey cam" / V-twin / drag-bike di saat idle dengan jitter retard ± skip-fire. Hanya aktif di RPM idle band — auto-off saat gas dibuka. Cocok motor sehari-hari yang mau sound character.

—

Bikin idle suara "lopey" / drag-bike feel via jitter retard ± skip-fire. Auto-disengage saat RPM keluar dari idle band (gas dibuka). Tidak aktif sebelum engine warm-up (default 60 detik dari boot).

⚠ Mode Aggressive & Drag Burble bisa stall engine kalau idle setting motor terlalu rendah — pastikan idle stable ≥1200 rpm dulu. Skip-fire akumulasi unburned fuel ke exhaust (lebih sedikit dari backfire).

[ 🔥 exhaust flame · rev-limit burner ]

Munculkan api keluar knalpot saat sustained di rev limiter (gas pol mentok limit ≥1 detik). Mekanisme: skip beberapa cycle (dump unburned fuel ke exhaust panas) + fire dengan retard ekstrem (combustion bocor ke exhaust stroke). Auto-disengage saat lepas gas / RPM turun.

idle 0 / 0 ms cooldown 0 ms

⚠ Cut mode HARD CUT tidak compatible — semua spark di-skip di rev-limit, tidak ada fire untuk ignite mixture di exhaust. Ganti ke Soft Retard atau Pattern Cut di panel "Rev Limiter" di atas.

⚠ Memang memperpendek umur exhaust valve & bisa naikkan EGT. Cocok untuk show / drag, BUKAN harian. Mode AGRESIF butuh konfirmasi tambahan karena risiko fuel wash + thermal shock.

▾ Detail teknis & safety gates

AMAN: pattern fire-fire-skip + retard 5-8° (jittered), engage delay 1 detik, max 3 detik aktif, cooldown 2 detik. Vbat min 11.5V.
AGRESIF: pattern fire-skip-skip + retard 12-20° (jittered), engage delay 1.5 detik, max 5 detik aktif, cooldown 5 detik. Vbat min 12V.

Auto-disengage: RPM turun di bawah limit-300, launch aktif, ALVP derate, spark disarm, panic button. Tidak engage sebelum uptime 30 detik (warmup proxy — exhaust harus panas dulu).

[ launch control · 2-step ]

Hold RPM di angka tetap saat tarik kopling (input GPIO13 LOW) — untuk start drag race. Tarik kopling, gas pol, RPM tertahan di angka yang di-set. Lepas kopling = launch.

Input GPIO13 (pull-up). Sambungkan ke saklar di tuas kopling / launch switch. Saat input LOW (kopling ditarik), pembatas rev override ke launch_rpm dengan HARD CUT.

[ quickshifter · clutchless up-shift ]

Cut pengapian X ms saat sensor strain di tuas perseneling trigger (GPIO14) → torque drop → gigi pindah tanpa kopling. Untuk shift up cepat di track. Perlu sensor QS terpasang di linkage.

Input GPIO14 (pull-up + interrupt FALLING). Sensor strain/microswitch di tuas perseneling. Trigger → cut ignition selama cut_ms → torsi drop → gigi pindah mulus.

[ shift light · gpio27 ]

LED/buzzer eksternal di GPIO27 yang nyala saat RPM mendekati limit. Warning di RPM warn (kedip), solid di RPM shift. Tanda visual untuk shift up tepat waktu — bantu tarik gigi sebelum touch rev limiter.

LED di GPIO27 mati di bawah warn, flash 5 Hz antara warn & shift, solid ON di atas shift. Sambungkan LED 3 mm + resistor 220 Ω dari GPIO27 ke GND.

[ kurva dwell · rpm-kompensasi ]

Adjust durasi dwell (waktu saturasi koil) berdasarkan RPM. Di RPM tinggi periode pendek → butuh dwell lebih singkat. Default 4 titik biasanya OK untuk Megapro stock; modif perlu tune sendiri.

Bila aktif, dwell µs dihitung dari curve di bawah berdasarkan RPM saat ini. Bila nonaktif, dwell fix dari tuning manual di card sebelah dipakai.

[ tuning output pengapian ]

Dwell = lama GPIO HIGH per fire (waktu coil saturasi). Offset advance = digeser ke seluruh map (kompensasi delay MOSFET + coil collapse). Strobe verify dengan timing light setelah tune.

Saat cranking (< ~1500 rpm) spark dipicu di edge CH2 (advance dasar mekanis tetap, period-independent) — mengatasi start susah seperti CDI stok. Default OFF. Berubah seketika & tersimpan ke NVS. WAJIB verifikasi strobe di mesin sebelum dipakai jalan — ini mengubah timing pengapian.

diagnostik pickup · EMI/anomali sinyal: — (0 = bersih · naik di bawah beban = pickup butuh shielding/ground)

Dwell = berapa lama GPIO25 HIGH per tembakan (= waktu saturasi primary coil dari MOSFET drive sebelum spark fires di fall edge). Offset menambah ke setiap lookup advance° — pakai untuk kompensasi propagation delay MOSFET + coil collapse (+ve = spark lebih cepat).

[ alvp · proteksi voltase rendah ]

Auto-protect saat aki lemah. Di voltase derate: rev limit dibatasi + dwell di-boost untuk kompensasi spark lemah. Di voltase disarm: spark dimatikan total agar koil tidak rusak. Aktifkan saat riding.

VBAT — V NORMAL

VBAT diukur via ADC GPIO32 + voltage divider 1:4 (mis. 30 kΩ + 10 kΩ). Di bawah derate threshold (default 10.5V), main rev limit auto-clamp ke derate_rpm (rider terasa power drop). Di bawah disarm threshold (default 9.0V), spark auto-disarm — koil HV butuh voltase cukup. Hysteresis 2 detik anti flicker saat cranking dip.

Tentang TDC reference & cara verify timing

TDC = Top Dead Center = posisi piston paling atas = titik 0° referensi. Semua advance diukur BTDC (Before TDC). Firmware tidak punya sensor TDC — yang ada cuma pulser. Hubungan fisik pickup → TDC ada di max advance ref°. 

   crank rotation  ──►
   ┌──────────────────────────────────────────────────┐
   │  CH1↓ ── CH2↓ ─── fire ──── ▲ TDC ── ATDC        │
   │   ▲       ▲        ▲        │                   │
   │  32°BTDC 14°BTDC  10°BTDC   0°                   │
   │ max_ref  base_ref   adv (dari map)               │
   └──────────────────────────────────────────────────┘
   (angka contoh Megapro stok)

Cara verify TDC dengan strobe light:

  1. Engine idle stabil ~1500 RPM, cabut sambungan ke koil sekunder kalau perlu.
  2. Pasang strobe light pickup ke kabel busi.
  3. Sinari flywheel — saat strobe nyala, lihat mark mana yang sejajar pointer:
    • T mark = fire AT TDC → map idle = 0° (terlalu retard, engine ngempos)
    • F mark = ~10° BTDC → factory idle Megapro, target ideal
    • PL mark = ~32° BTDC → peak advance saat rev tinggi
  4. Mismatch antara map setting vs actual strobe reading?
    • Kalau bias konsisten (e.g., semua lambat 3°) → adjust offset advance di panel Tuning Output Pengapian.
    • Kalau peak mismatch (32° di map tapi strobe baca 28°) → adjust max advance ref di sini.

Kapan perlu edit manual:

  • Modif rotor magnetic (pulser dipindah / dibuat ulang custom).
  • Ganti pickup ke geometry beda (jarak ke flywheel, sudut mount).
  • Motor non-standard yang tidak ada preset-nya.

⚙ Edit manual (override mode = user):

base advance ref = max − magnet (otomatis hitung).
Cara ukur magnet width: capture pulser di scope mode, ukur waktu CH1 fall → CH2 fall × 360° / periode CH1.
Cara cek max advance: strobe timing light di TDC mark saat engine ramp ke high RPM — pembacaan max di angka peak advance.

⚠ Salah set = timing seluruh map melenceng → potensi kickback / detonation. Strobe verify di TDC mark setelah apply.
